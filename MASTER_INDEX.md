## 🚀 QUICK COMMANDS

### **Setup**
```bash
# Copy environment file
cp .env.example .env.local

# Install dependencies
npm install

# Run development server
npm run dev
```

### **Diagnostic**
```bash
# Check connection
node check-supabase-connection.js

# Test CRUD
node test-database.js

# Auto setup
bash setup-database.sh
```

### **Troubleshooting**
```bash
# Check environment
cat .env.local

# Check logs
npm run dev

# Restart server
# Ctrl+C, then:
npm run dev
```
